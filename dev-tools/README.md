# Інструменти розробки / Development Tools

Ця папка містить інструменти для розробки, налагодження та підтримки проекту:

## Аналіз та діагностика
- `analyze_colors.py` - аналіз кольорів на зображеннях
- `analyze_table.py` - аналіз таблиць розкладів
- `check_*.py` - перевірка різних компонентів системи
- `inspect_db.py`, `db_inspector.py` - інструменти інспекції бази даних

## Налаштування та конфігурація
- `manual_setup.py` - ручне налаштування системи
- `setup_table.py` - налаштування меж таблиць
- `clear_*.py` - очищення даних

## Тести та експерименти
- `test.py` - основний тестовий файл
- `test_cache.py` - тест кешування
- `test_colors.py` - тест кольорів

## Документація
- `DB_INSPECTOR_README.md` - документація по інспектору БД
- `OCR_SETUP_README.md` - налаштування OCR
- `copilot-readme.md` - документація для Copilot

## Використання
```bash
# Перевірка бази даних
python dev-tools/inspect_db.py

# Очищення тестових даних
python dev-tools/clear_db.py
```
