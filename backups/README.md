# Резервні копії / Backups

Ця папка містить резервні копії файлів та альтернативні версії:

## Резервні версії бота
- `bot backup.py` - резервна копія оригінального бота
- `bot ocr model.py` - версія бота з OCR моделлю

## Серверні файли
- `server_bot.py` - серверна версія бота
- `serverlogs.txt` - логи серверної версії
- `server_users.db` - база даних серверної версії

## Важливо!
Ці файли зберігаються для:
- Відновлення функціональності при проблемах
- Порівняння різних підходів до реалізації
- Історичної довідки про розвиток проекту

**Не видаляйте цю папку без потреби!**
