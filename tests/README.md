# Тестові файли / Test Files

Ця папка містить всі тестові файли, розділені на категорії:

## unit/
Модульні тести для окремих компонентів:
- `test_formatting*.py` - тести форматування повідомлень
- `test_ocr*.py` - тести розпізнавання тексту
- `test_parser*.py` - тести парсингу даних
- `test_toggle*.py` - тести функцій перемикання налаштувань

## integration/
Інтеграційні тести для перевірки взаємодії компонентів:
- `test_monitor*.py` - тести моніторингу сайтів
- `test_today.py` - тест відображення графіка на сьогодні
- `test_stars.py` - тест функціональності зірок

## Запуск тестів
```bash
# Запуск всіх unit тестів
python -m pytest tests/unit/ -v

# Запуск конкретного тесту
python tests/unit/test_formatting.py
```
