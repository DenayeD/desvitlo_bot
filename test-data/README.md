# Тестові дані / Test Data

Ця папка містить тестові дані та файли для розробки та тестування:

## Бази даних
- `test_users.db` - тестова база даних користувачів

## Зображення
- `test_schedule.png` - тестове зображення розкладу

## Конфігураційні файли
- `manual_table_settings.json` - ручні налаштування таблиць
- `table_bounds.json` - межі таблиць для OCR

## Папки
- `test_clocks/` - тестові зображення годинників
- `tests/` - додаткова папка з тестами (якщо була)

## Використання
Ці файли використовуються тестовими скриптами в папках `tests/` та `dev-tools/` для перевірки функціональності системи.
